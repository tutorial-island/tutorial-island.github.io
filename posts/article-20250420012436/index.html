Implementing a Simple Palindrome Checker in JavaScript

Apr 20, 2025· Tutorial Island

Introduction

Today, we’re going to implement a simple palindrome checker in JavaScript. A palindrome is a word, phrase, number, or other sequence of characters which reads the same backward as forward (ignoring spaces, punctuation, and capitalization). Our goal is to create a function that takes a string as input and returns true if it’s a palindrome and false otherwise.

Explanation

The challenge lies in correctly comparing the input string to its reversed version. This involves several considerations:

  1. Case Insensitivity: Palindromes should be case-insensitive (e.g., “Racecar” is a palindrome). We need to convert the input string to either lowercase or uppercase for consistent comparison.
  2. Non-alphanumeric Characters: Spaces, punctuation, and other non-alphanumeric characters should be ignored. We need to filter these out from the string before comparison.
  3. String Reversal: We need to efficiently reverse the string. JavaScript offers several ways to achieve this.

We will leverage the following JavaScript functionalities:

  • .toLowerCase(): Converts a string to lowercase.
  • .replace(/[^a-z0-9]/gi, ''): Removes non-alphanumeric characters using a regular expression. [^a-z0-9] matches any character that is not a letter (a-z) or a number (0-9). The g flag means “global” (replace all matches) and the i flag means “case-insensitive”.
  • .split(''): Splits a string into an array of characters.
  • .reverse(): Reverses the order of elements in an array.
  • .join(''): Joins the elements of an array into a string.

Requirements

  • The function should accept a string as input.
  • The function should return true if the string is a palindrome (case-insensitive, ignoring non-alphanumeric characters) and false otherwise.
  • The function should handle empty strings and single-character strings correctly.

Solutions

function cleanString(str) {
    // Remove non-alphanumeric characters and convert to lowercase
    const cleanedStr = str.replace(/[^a-z0-9]/gi, '').toLowerCase();
    return cleanedStr;
}

// Assertions to verify the cleaning process
console.assert(cleanString("Racecar!") === "racecar", "Test Case 1 Failed: Cleaning with special characters");
console.assert(cleanString("A man, a plan, a canal: Panama") === "amanaplanacanalpanama", "Test Case 2 Failed: Cleaning with multiple special characters and case");

The cleanString function focuses on pre-processing the input string. It uses a regular expression /[^a-z0-9]/gi to remove any character that isn’t a letter or number. Converting the string to lowercase using .toLowerCase() ensures case-insensitive comparison later. This step isolates the core alphanumeric characters, crucial for accurate palindrome detection.

function reverseString(str) {
  // Reverse a string
  const reversedStr = str.split('').reverse().join('');
  return reversedStr;
}

// Assertions to verify the string reversing process
console.assert(reverseString("racecar") === "racecar", "Test Case 1 Failed: Reversing racecar");
console.assert(reverseString("hello") === "olleh", "Test Case 2 Failed: Reversing hello");

The reverseString function tackles the string reversal requirement. First, we use .split('') to break the string into an array of individual characters. Then, .reverse() reverses the order of elements within the array. Finally, .join('') concatenates the reversed characters back into a single string. This provides a clear and concise way to obtain the reversed version of any given string.

function isPalindrome(str) {
  // Check if a string is a palindrome
  const cleaned = cleanString(str);
  const reversed = reverseString(cleaned);
  return cleaned === reversed;
}

// Assertions to verify the palindrome check
console.assert(isPalindrome("Racecar") === true, "Test Case 1 Failed: Racecar");
console.assert(isPalindrome("A man, a plan, a canal: Panama") === true, "Test Case 2 Failed: Long Phrase");
console.assert(isPalindrome("hello") === false, "Test Case 3 Failed: hello");
console.assert(isPalindrome("") === true, "Test Case 4 Failed: Empty String");
console.assert(isPalindrome("A") === true, "Test Case 5 Failed: Single Character String");

The isPalindrome function implements the core palindrome logic. It first calls cleanString to pre-process the input, removing irrelevant characters and handling case insensitivity. Then, it utilizes the reverseString function to obtain the reversed version of the cleaned string. Finally, it directly compares the cleaned string to its reversed counterpart, returning true if they are identical (indicating a palindrome) and false otherwise.

Thoughts

In this tutorial, we dissected the palindrome checking problem into smaller, manageable parts. We addressed the challenges of case insensitivity and non-alphanumeric characters by cleaning the input string. Then we efficiently reversed the cleaned string. Finally, we compared the cleaned string to its reversed version to determine if it was a palindrome. By breaking down the problem into functions with clear responsibilities, we created a robust and readable solution.

function cleanString(str) {
    // Remove non-alphanumeric characters and convert to lowercase
    const cleanedStr = str.replace(/[^a-z0-9]/gi, '').toLowerCase();

}

function reverseString(str) {
  // Reverse a string
  const reversedStr = str.split('').reverse().join('');
  return reversedStr;
}

function isPalindrome(str) {
  // Check if a string is a palindrome
  const cleaned = cleanString(str);
  const reversed = reverseString(cleaned);
  return cleaned === reversed;
}

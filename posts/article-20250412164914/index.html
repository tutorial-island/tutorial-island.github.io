Implementing a Simple Palindrome Checker in JavaScript

Apr 12, 2025· Tutorial Island

Introduction

Today, we’re diving into a classic problem: building a palindrome checker in JavaScript. A palindrome is a word, phrase, number, or other sequence of characters which reads the same backward as forward (ignoring spaces, punctuation, and capitalization). Examples include “madam,” “racecar,” and “A man, a plan, a canal: Panama.” We’ll write a function that determines whether a given string is a palindrome.

Explanation

The challenge with palindrome checking lies in pre-processing the input string. We need to:

  1. Remove non-alphanumeric characters: Spaces, punctuation, and other symbols should be ignored.
  2. Convert to lowercase: Palindrome checks should be case-insensitive (“Racecar” is a palindrome).
  3. Compare the processed string with its reverse: This is the core logic for determining if it’s a palindrome.

To accomplish this, we’ll use the following JavaScript functionalities:

  • .replace() with regular expressions: To remove non-alphanumeric characters. The regex /[^a-z0-9]/gi will match any character that isn’t a letter (a-z) or a number (0-9), globally (g) and case-insensitively (i).
  • .toLowerCase(): To convert the string to lowercase.
  • .split(''), .reverse(), and .join(''): To reverse the string. .split('') turns the string into an array of characters, .reverse() reverses the array, and .join('') joins the characters back into a string.

Requirements

  • Functional Requirements:
    • The function must correctly identify palindromes, ignoring case and non-alphanumeric characters.
    • The function should return a boolean value: true if the input is a palindrome, false otherwise.
  • Non-Functional Requirements:
    • The code should be readable and well-commented.
    • The function should handle empty strings and strings containing only non-alphanumeric characters gracefully (treating them as palindromes).

Solutions

function cleanString(str) {
  // Remove non-alphanumeric characters and convert to lowercase.
  const cleanedStr = str.replace(/[^a-z0-9]/gi, '').toLowerCase();
  return cleanedStr;
}

// Assert statements for the cleanString function
console.assert(cleanString("A man, a plan, a canal: Panama") === "amanaplanacanalpanama", "Test Case 1 Failed: Cleaning string with special characters");
console.assert(cleanString("Racecar") === "racecar", "Test Case 2 Failed: Cleaning a simple palindrome");

This cleanString function pre-processes the input string by removing any characters that aren’t letters or numbers using a regular expression and then converts the string to lowercase. This ensures that the palindrome check is case-insensitive and ignores irrelevant characters. We are using replace method because its the more appropriate method to replace charaters in a string given a regexp. The regular expression [^a-z0-9] is a character class that matches any character that is NOT (denoted by ^ inside the brackets) a lowercase letter from a to z or a digit from 0 to 9. The g flag ensures that all occurrences are replaced, not just the first one. The i flag makes the match case-insensitive.

function reverseString(str) {
    // Reverse a string.
    return str.split('').reverse().join('');
}

// Assert statements for the reverseString function
console.assert(reverseString("hello") === "olleh", "Test Case 1 Failed: Reversing 'hello'");
console.assert(reverseString("") === "", "Test Case 2 Failed: Reversing an empty string");

The reverseString function reverses the input string. It first splits the string into an array of characters using .split(''), then reverses the order of elements in the array using .reverse(), and finally joins the elements back into a string using .join(''). This reversed string will then be compared to the cleaned string to determine if it’s a palindrome.

function isPalindrome(str) {
  // Check if a string is a palindrome.
  const cleanedStr = cleanString(str);
  const reversedStr = reverseString(cleanedStr);

  return cleanedStr === reversedStr;
}

// Assert statements for the isPalindrome function
console.assert(isPalindrome("A man, a plan, a canal: Panama") === true, "Test Case 1 Failed: Complex palindrome");
console.assert(isPalindrome("racecar") === true, "Test Case 2 Failed: Simple palindrome");
console.assert(isPalindrome("hello") === false, "Test Case 3 Failed: Non-palindrome");
console.assert(isPalindrome("") === true, "Test Case 4 Failed: Empty string");
console.assert(isPalindrome(".,") === true, "Test Case 5 Failed: Punctuation only");

The isPalindrome function checks if a given string is a palindrome. First, it cleans the input string using the cleanString function, removing non-alphanumeric characters and converting it to lowercase. Then, it reverses the cleaned string using the reverseString function. Finally, it compares the cleaned string with its reversed version. If they are the same, it returns true (indicating that the original string is a palindrome); otherwise, it returns false.

Thoughts

We’ve successfully built a palindrome checker in JavaScript. The key steps were:

  1. Cleaning the Input: Removing irrelevant characters and converting to lowercase.
  2. Reversing the String: Creating a reversed version of the cleaned string.
  3. Comparison: Comparing the cleaned string with its reversed version.

Here’s the complete code:

function cleanString(str) {
  // Remove non-alphanumeric characters and convert to lowercase.
  const cleanedStr = str.replace(/[^a-z0-9]/gi, '').toLowerCase();
  return cleanedStr;
}

function reverseString(str) {
    // Reverse a string.
    return str.split('').reverse().join('');
}

function isPalindrome(str) {
  // Check if a string is a palindrome.
  const cleanedStr = cleanString(str);
  const reversedStr = reverseString(cleanedStr);

  return cleanedStr === reversedStr;
}

// Assert statements for the functions
console.assert(cleanString("A man, a plan, a canal: Panama") === "amanaplanacanalpanama", "Test Case 1 Failed: Cleaning string with special characters");
console.assert(cleanString("Racecar") === "racecar", "Test Case 2 Failed: Cleaning a simple palindrome");
console.assert(reverseString("hello") === "olleh", "Test Case 3 Failed: Reversing 'hello'");
console.assert(reverseString("") === "", "Test Case 4 Failed: Reversing an empty string");
console.assert(isPalindrome("A man, a plan, a canal: Panama") === true, "Test Case 5 Failed: Complex palindrome");
console.assert(isPalindrome("racecar") === true, "Test Case 6 Failed: Simple palindrome");
console.assert(isPalindrome("hello") === false, "Test Case 7 Failed: Non-palindrome");
console.assert(isPalindrome("") === true, "Test Case 8 Failed: Empty string");
console.assert(isPalindrome(".,") === true, "Test Case 9 Failed: Punctuation only");

This approach is efficient and readable, making it a good solution for palindrome checking in JavaScript. Remember that the key is breaking down the problem into smaller, manageable functions.

Implementing a Simple Palindrome Checker in Kotlin

Apr 25, 2025· Tutorial Island

Introduction

Today, we are going to implement a simple palindrome checker in Kotlin. A palindrome is a word, phrase, number, or other sequence of characters which reads the same backward or forward. Our goal is to write a function that can determine whether a given string is a palindrome, ignoring case and non-alphanumeric characters.

Explanation

The challenge of building a palindrome checker involves several considerations:

  1. Case Sensitivity: Palindromes should be case-insensitive. “Racecar” is a palindrome, and so is “racecar”. We need to convert the input string to either lowercase or uppercase.
  2. Non-Alphanumeric Characters: Spaces, punctuation, and other non-alphanumeric characters should be ignored. “A man, a plan, a canal: Panama” is a palindrome if these characters are removed. We’ll need to filter these out.
  3. String Reversal: We need an efficient way to compare the original string (after cleaning) with its reversed version.

To tackle these challenges, we’ll use the following Kotlin features:

  • .toLowerCase(): For converting the string to lowercase.
  • .filter { it.isLetterOrDigit() }: For removing non-alphanumeric characters. This filters the string, keeping only characters that are letters or digits.
  • .reversed(): For reversing the cleaned string.
  • == operator: For comparing the cleaned string with its reversed version.

Requirements

  • Functional Requirements:
    • The function must accept a string as input.
    • The function must return true if the input string is a palindrome (ignoring case and non-alphanumeric characters), and false otherwise.
  • Non-Functional Requirements:
    • The code should be readable and well-commented.
    • The solution should be reasonably efficient.

Solutions

fun cleanString(input: String): String {
    val cleanedString = input.toLowerCase().filter { it.isLetterOrDigit() }
    return cleanedString
}

This function cleanString takes a string as input, converts it to lowercase, and removes any non-alphanumeric characters. This ensures that the palindrome check is case-insensitive and ignores irrelevant characters. We use .toLowerCase() for case insensitivity and .filter { it.isLetterOrDigit() } to iterate over each character of the input and checks if is a Letter or a Digit.

fun isPalindrome(input: String): Boolean {
    val cleanedString = cleanString(input)
    val reversedString = cleanedString.reversed()
    return cleanedString == reversedString
}

fun main() {
    // Test cases
    assert(isPalindrome("Racecar"))
    assert(isPalindrome("A man, a plan, a canal: Panama"))
    assert(!isPalindrome("hello"))
}

The isPalindrome function takes a string as input, cleans it using the cleanString function we created, reverses the cleaned string, and then compares the cleaned string with its reversed version. If they are equal, the function returns true; otherwise, it returns false. This implements the core logic of the palindrome checker.

Thoughts

In this article, we built a palindrome checker in Kotlin that is both case-insensitive and ignores non-alphanumeric characters. We began by identifying the challenges: case sensitivity, handling irrelevant characters, and efficiently comparing the string with its reverse. We then used Kotlin’s string manipulation functions like .toLowerCase(), .filter { it.isLetterOrDigit() }, and .reversed() to address these challenges. Finally, we combined these components into a single isPalindrome function and verified its correctness with assert statements.

Here’s the complete solution as a single code snippet:

fun cleanString(input: String): String {
    val cleanedString = input.toLowerCase().filter { it.isLetterOrDigit() }
    return cleanedString
}

fun isPalindrome(input: String): Boolean {
    val cleanedString = cleanString(input)
    val reversedString = cleanedString.reversed()
    return cleanedString == reversedString
}

fun main() {
    // Test cases
    assert(isPalindrome("Racecar"))
    assert(isPalindrome("A man, a plan, a canal: Panama"))
    assert(!isPalindrome("hello"))
    assert(isPalindrome("Madam"))
}

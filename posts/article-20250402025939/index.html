Building a Simple Word Counter in Go

Oct 27, 2023· Bard

Introduction

Today, we’re going to implement a simple word counter in Go. The program will take a string as input and return a map containing each unique word in the string and the number of times it appears. This is a common task in text analysis and data processing.

Explanation

The challenge here is to efficiently parse the input string, identify individual words, and keep track of their counts. We’ll need to handle:

  • String Splitting: The input string needs to be broken down into individual words. We need a way to separate words based on spaces or other delimiters.
  • Case Insensitivity: We might want to treat “The” and “the” as the same word. We can achieve this by converting the words to lowercase.
  • Data Structure for Counts: We need a data structure to store the word counts. A map (dictionary) in Go is perfect for this, where the key is the word (string) and the value is the count (integer).

We’ll be using the following Go functionalities:

  • strings.Fields(s string) []string: This function splits a string s into a slice of strings, using spaces as the delimiter.
  • strings.ToLower(s string) string: This function converts a string s to lowercase.
  • map[string]int: Go’s built-in map type to store word counts.
  • testing package: Go’s built-in library for writing unit tests.

Requirements

Functional Requirements:

  1. The program must take a string as input.
  2. The program must return a map where the keys are unique words from the input string, and the values are the number of times each word appears.
  3. The program should handle case-insensitivity (count “The” and “the” as the same word).
  4. The program should handle strings with multiple spaces between words.
  5. The program must include unit tests to verify its correctness.

Non-Functional Requirements:

  1. The code should be readable and well-commented.
  2. The code should be efficient.

Solutions

package main

import (
	"strings"
	"testing"
)

// Function to clean a single word by removing leading/trailing punctuation and converting to lowercase
func cleanWord(word string) string {
	word = strings.Trim(word, ".,?!-;:") // Remove punctuation
	return strings.ToLower(word)        // Convert to lowercase
}

// Test cleaning the word
func TestCleanWord(t *testing.T) {
	if cleanWord("Hello!") != "hello" {
		t.Errorf("Expected 'hello', got '%s'", cleanWord("Hello!"))
	}
	if cleanWord("World.") != "world" {
		t.Errorf("Expected 'world', got '%s'", cleanWord("World."))
	}
}

// countWords takes a string as input and returns a map of word counts.
func countWords(input string) map[string]int {
	wordCounts := make(map[string]int)  // Initialize an empty map to store word counts.
	words := strings.Fields(input)       // Split the input string into a slice of words.

	for _, word := range words {           // Iterate over each word in the slice.
		cleanedWord := cleanWord(word)
		wordCounts[cleanedWord]++        // Increment the count for the word in the map.
	}

	return wordCounts                     // Return the map of word counts.
}

// Test the word counter function
func TestCountWords(t *testing.T) {
	input := "The quick brown fox jumps over the lazy dog."
	expected := map[string]int{
		"the":   2,
		"quick": 1,
		"brown": 1,
		"fox":   1,
		"jumps": 1,
		"over":  1,
		"lazy":  1,
		"dog":   1,
	}
	result := countWords(input)

	// Check if the result matches the expected map.
	for word, count := range expected {
		if result[word] != count {
			t.Errorf("For word '%s', expected count %d, got %d", word, count, result[word])
		}
	}

	input2 := "This is a test, is it?"
	expected2 := map[string]int{
		"this": 1,
		"is":   2,
		"a":    1,
		"test": 1,
		"it":   1,
	}

	result2 := countWords(input2)
	for word, count := range expected2 {
		if result2[word] != count {
			t.Errorf("For word '%s', expected count %d, got %d", word, count, result2[word])
		}
	}

}

func main() {
	// Example usage (not run during tests)
	text := "The quick brown fox jumps over the lazy dog. The dog is lazy."
	counts := countWords(text)
	for word, count := range counts {
		println(word, ":", count)
	}
}

We initialize the map using make(map[string]int). strings.Fields(input) splits the string into words using spaces as separators. We then iterate through the words, converting each to lowercase using the cleanWord function before incrementing the count in the map.

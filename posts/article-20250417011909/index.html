Implementing a Simple Text Analyzer in Python

Apr 17, 2025· Tutorial Island

Introduction

Today, we are going to implement a simple text analyzer. This analyzer will take a string of text as input and determine the frequency of each word within that text. This is a foundational exercise that touches on several core Python concepts.

Explanation

The primary challenge in this problem lies in efficiently counting the occurrences of each word. We need to:

  1. Clean the Text: Remove punctuation and convert the text to lowercase to ensure that “The” and “the” are treated as the same word.
  2. Split the Text: Divide the text into individual words.
  3. Count the Words: Store and update the count of each word. We can use a dictionary to achieve this, where the word is the key and the count is the value.
  4. Handle Edge Cases: Consider how to deal with empty input strings or strings with only punctuation.

We will be utilizing the following Python functionalities:

  • .lower(): To convert the text to lowercase.
  • .split(): To split the text into a list of words.
  • string.punctuation: To easily access a string containing all punctuation characters for removal.
  • Dictionaries: To store and access word counts efficiently.
  • for loops: To iterate through the string and clean out any noise

Requirements

Functional Requirements:

  • The analyzer must accept a string as input.
  • The analyzer must return a dictionary where keys are words (lowercase) and values are their frequencies.
  • Punctuation should be removed from the text before analysis.
  • The analyzer should handle empty input strings gracefully, returning an empty dictionary.

Non-Functional Requirements:

  • The code should be readable and well-commented.
  • The solution should be efficient for relatively small text inputs.

Solutions

import string

def clean_text(text):
    """
    Removes punctuation and converts text to lowercase.

    Args:
        text: The input string.

    Returns:
        The cleaned string.
    """
    text = text.lower()
    text = text.translate(str.maketrans('', '', string.punctuation))
    return text

assert clean_text("Hello, World!") == "hello world"
assert clean_text("This is a test.") == "this is a test"

This function clean_text prepares the input text for analysis. First, we convert the entire string to lowercase using .lower(). This ensures that words like “The” and “the” are counted together. Then, we remove punctuation using str.maketrans('', '', string.punctuation) to create a translation table and .translate() to apply it. This step ensures that punctuation marks don’t affect word counts (e.g., “world” and “world!” are treated as the same word).

def count_word_frequencies(text):
    """
    Counts the frequency of each word in a string.

    Args:
        text: The input string.

    Returns:
        A dictionary where keys are words and values are their frequencies.
    """
    text = clean_text(text)
    words = text.split()
    word_counts = {}

    for word in words:
        if word: # check for possible spaces that generated empty strings
            if word in word_counts:
                word_counts[word] += 1
            else:
                word_counts[word] = 1

    return word_counts

assert count_word_frequencies("the quick brown fox jumps over the lazy dog") == {'the': 2, 'quick': 1, 'brown': 1, 'fox': 1, 'jumps': 1, 'over': 1, 'lazy': 1, 'dog': 1}
assert count_word_frequencies("This is a test. This is only a test!") == {'this': 2, 'is': 2, 'a': 2, 'test': 2, 'only': 1}
assert count_word_frequencies("") == {}

The count_word_frequencies function orchestrates the entire process. First, it calls clean_text to prepare the input. Then, it splits the cleaned text into a list of words using .split(). We initialize an empty dictionary word_counts to store the word frequencies. The for loop iterates through each word in the words list. Inside the loop, we check if the word is already a key in the word_counts dictionary. If it is, we increment its count; otherwise, we add it to the dictionary with a count of 1. The final dictionary, containing the word frequencies, is then returned. An initial if word: is added as a guard against possible spaces added to the string creating empty strings that could become keys.

Thoughts

The solution presented breaks down the problem into smaller, manageable parts. clean_text handles the crucial task of normalizing the input text, making the subsequent word counting more accurate. count_word_frequencies then iterates through the normalized text and builds the frequency map. The assertions ensure the core functionality works as expected. This approach makes the code easier to understand, test, and maintain.

    Removes punctuation and converts text to lowercase.

        The cleaned string.
    """

    return text

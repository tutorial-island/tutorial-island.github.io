Kotlin: Implementing a Simple Palindrome Checker

Apr 21, 2024· Tutorial Island

Introduction

Today, we’re going to implement a simple palindrome checker in Kotlin. A palindrome is a word, phrase, number, or other sequence of characters which reads the same backward or forward. For instance, “madam” or “racecar” are palindromes. Our goal is to create a function that takes a string as input and returns true if it’s a palindrome, and false otherwise.

Explanation

The challenge in implementing a palindrome checker lies in comparing the original string with its reversed version. We need to handle cases where the string contains uppercase and lowercase letters, and potentially ignore spaces or punctuation.

To solve this, we’ll use the following Kotlin functionalities:

  • String manipulation: We’ll use functions like .toLowerCase() to convert the string to lowercase for case-insensitive comparison, and .filter { it.isLetterOrDigit() } to remove non-alphanumeric characters.
  • String reversal: We’ll use the .reversed() function to create the reversed version of the processed string.
  • Conditional logic: We’ll use if statements or the equality operator (==) to compare the processed string with its reversed version.
  • Testing: We’ll use assert statements to verify our function works correctly with various inputs.

Requirements

  • Functional Requirements:
    • The function must take a string as input.
    • The function must return true if the input string is a palindrome (case-insensitive, ignoring non-alphanumeric characters), and false otherwise.
  • Non-Functional Requirements:
    • The code should be readable and well-commented.
    • The function should handle empty strings and strings with only non-alphanumeric characters gracefully.

Solutions

1. Remove non-alphanumeric characters and convert to lowercase:

fun cleanString(input: String): String {
    val cleanedString = input.toLowerCase().filter { it.isLetterOrDigit() }
    return cleanedString
}

fun main() {
    assert(cleanString("Racecar") == "racecar")
    assert(cleanString("A man, a plan, a canal: Panama") == "amanaplanacanalpanama")
}

This function, cleanString, takes a string as input and first converts it to lowercase using .toLowerCase(). Then, it filters the string using .filter { it.isLetterOrDigit() }, keeping only alphanumeric characters. This ensures that case and non-alphanumeric characters don’t affect the palindrome check. The assert statements verify that the function correctly cleans the strings “Racecar” and “A man, a plan, a canal: Panama.”

2. Reverse the cleaned string:

fun reverseString(input: String): String {
    return input.reversed()
}

fun main() {
    assert(reverseString("racecar") == "racecar")
    assert(reverseString("hello") == "olleh")
}

This function, reverseString, simply reverses the input string using the .reversed() function. This is a straightforward way to obtain the reversed version of a string in Kotlin, which we’ll then compare to the original (cleaned) string. The assert statements confirm the function correctly reverses “racecar” and “hello.”

3. Check if the cleaned string is equal to its reversed version:

fun isPalindrome(input: String): Boolean {
    val cleaned = cleanString(input)
    val reversed = reverseString(cleaned)
    return cleaned == reversed
}

fun main() {
    assert(isPalindrome("racecar"))
    assert(!isPalindrome("hello"))
}

This function, isPalindrome, combines the previous two functions. First it cleans the input with cleanString function from the step 1, then reverses the cleaned string using the reverseString function from the step 2. Finally, it compares the cleaned string to its reversed version. If they are equal, the function returns true, indicating that the input string is a palindrome. The assert statements demonstrate this with “racecar” (a palindrome) and “hello” (not a palindrome).

Thoughts

Here’s the complete solution, combining all the functions and adding more comprehensive tests:

fun cleanString(input: String): String {
    val cleanedString = input.toLowerCase().filter { it.isLetterOrDigit() }
    return cleanedString
}

fun reverseString(input: String): String {
    return input.reversed()
}

fun isPalindrome(input: String): Boolean {
    val cleaned = cleanString(input)
    val reversed = reverseString(cleaned)
    return cleaned == reversed
}

fun main() {
    assert(isPalindrome("Racecar"))
    assert(isPalindrome("A man, a plan, a canal: Panama"))
    assert(isPalindrome("Madam"))
    assert(!isPalindrome("hello"))
    assert(isPalindrome("")) // Empty string should be a palindrome
    assert(isPalindrome(" ")) //String with only spaces is a palindrome
    assert(isPalindrome("121"))
    assert(!isPalindrome("123"))
}

This complete solution encapsulates the logic for cleaning the input string, reversing it, and then comparing the two. The added assert statements in main() cover a wider range of test cases, including empty strings, strings with only spaces, and numerical palindromes, ensuring the isPalindrome function is robust and reliable. By breaking down the problem into smaller, manageable functions, we’ve created a readable and maintainable palindrome checker in Kotlin. This example demonstrates good practices such as code clarity, modularity, and thorough testing, making it a valuable learning experience for junior developers.

Implementing a Simple Shopping Cart with TypeScript

Oct 27, 2023· Tutorial Island

Introduction

Today, we’re going to build a rudimentary shopping cart system using TypeScript. This will involve adding items to the cart, removing items, and calculating the total cost. This exercise will help you understand how to effectively use TypeScript to manage data structures and ensure type safety in a practical scenario.

Explanation

The challenge in building a shopping cart lies in managing the items added and their quantities. We need a way to store this information, ensure that only valid products are added, and accurately calculate the total price.

Here are the key challenges we’ll address:

  • Data Structure: We need to choose a suitable data structure to store the items in the cart, associating each item with its quantity. A simple array won’t cut it; we need key-value pairs.
  • Type Safety: TypeScript’s strength lies in type safety. We want to define strict types for products and the shopping cart to prevent errors like adding non-existent products or calculating prices with incorrect types.
  • Functionality: We need to implement functions to add items, remove items, and calculate the total. These functions should be robust and handle edge cases.

We will leverage the following TypeScript features:

  • Interfaces: To define the structure of a Product.
  • Type Aliases: Potentially to define the type of the shopping cart itself.
  • Objects: To represent the shopping cart itself and individual products.
  • Functions: To encapsulate the core operations of adding, removing, and calculating the total.

Requirements

Here’s a summary of the functional requirements for our shopping cart:

  • Add Items: A function to add products to the cart with a specified quantity.
  • Remove Items: A function to remove products from the cart.
  • Calculate Total: A function to calculate the total cost of all items in the cart.
  • Data Integrity: Ensure only valid products can be added, and quantities are always positive integers.

Solutions

Here are some code snippets demonstrating how to fulfill the requirements:

interface Product {
    id: string;
    name: string;
    price: number;
}

type Cart = {
    [productId: string]: {
        product: Product;
        quantity: number;
    };
};

/**
 * Adds a product to the cart or updates the quantity if it already exists.
 * @param cart The shopping cart object.
 * @param product The product to add.
 * @param quantity The quantity to add (must be a positive integer).
 * @returns The updated shopping cart.
 */
function addItemToCart(cart: Cart, product: Product, quantity: number): Cart {
    if (quantity <= 0) {
        throw new Error("Quantity must be a positive integer.");
    }

    const existingItem = cart[product.id];

    if (existingItem) {
        cart[product.id].quantity += quantity;
    } else {
        cart[product.id] = { product: product, quantity: quantity };
    }

    return cart;
}

// Assert statements to verify the function's behavior
const initialCart: Cart = {};
const product1: Product = { id: "1", name: "Laptop", price: 1200 };
const updatedCart = addItemToCart(initialCart, product1, 2);
console.assert(updatedCart["1"].quantity === 2, "Adding item to cart failed");
console.assert(updatedCart["1"].product.name === "Laptop", "Product name is incorrect");

This function, addItemToCart, takes the cart, product, and quantity as input. It first validates the quantity to ensure it’s a positive integer. If the product already exists in the cart, it updates the quantity. Otherwise, it adds the product to the cart with the specified quantity. We’re using an object (Cart) where the product ID serves as the key, allowing for quick lookup and update of items.

/**
 * Removes a product from the cart.
 * @param cart The shopping cart object.
 * @param productId The ID of the product to remove.
 * @returns The updated shopping cart.
 */
function removeItemFromCart(cart: Cart, productId: string): Cart {
    if (cart[productId]) {
        delete cart[productId];
    }

    return cart;
}

// Assert statements to verify the function's behavior
const cartWithItem: Cart = {
    "1": { product: { id: "1", name: "Laptop", price: 1200 }, quantity: 1 },
};
const updatedCartAfterRemoval = removeItemFromCart(cartWithItem, "1");
console.assert(Object.keys(updatedCartAfterRemoval).length === 0, "Removing item from cart failed");

The removeItemFromCart function removes a product from the cart based on its ID. It checks if the product exists in the cart before attempting to remove it using the delete operator. The assert statement confirms that the item is indeed removed, resulting in an empty cart (as it was the only item).

/**
 * Calculates the total cost of all items in the cart.
 * @param cart The shopping cart object.
 * @returns The total cost.
 */
function calculateTotal(cart: Cart): number {
    let total = 0;
    for (const productId in cart) {
        if (cart.hasOwnProperty(productId)) {
            total += cart[productId].product.price * cart[productId].quantity;
        }
    }
    return total;
}

// Assert statements to verify the function's behavior
const cartWithTwoItems: Cart = {
    "1": { product: { id: "1", name: "Laptop", price: 1200 }, quantity: 1 },
    "2": { product: { id: "2", name: "Mouse", price: 25 }, quantity: 2 },
};
const totalCost = calculateTotal(cartWithTwoItems);
console.assert(totalCost === 1250, "Calculating total failed");

The calculateTotal function iterates through the cart, summing the price of each item multiplied by its quantity. The hasOwnProperty check ensures that we’re only iterating over the cart’s own properties and not properties inherited from its prototype. The assert statement verifies that the total cost is calculated correctly based on the items in the cart.

Thoughts

Let’s consolidate these functions into a single, complete shopping cart implementation:

    id: string;
    name: string;
    price: number;
}

type Cart = {
    [productId: string]: {
        product: Product;
        quantity: number;
    };
};

    }

    return cart;
}

function removeItemFromCart(cart: Cart, productId: string): Cart {
    if (cart[productId]) {
        delete cart[productId];
    }

    return cart;

    for (const productId in cart) {
        if (cart.hasOwnProperty(productId)) {
            total += cart[productId].product.price * cart[productId].quantity;

// Example Usage and Assertions
const product1: Product = { id: "1", name: "Laptop", price: 1200 };
const product2: Product = { id: "2", name: "Mouse", price: 25 };

let myCart: Cart = {};

myCart = addItemToCart(myCart, product1, 1);
myCart = addItemToCart(myCart, product2, 2);

console.assert(myCart["1"].quantity === 1, "Adding product1 failed");
console.assert(myCart["2"].quantity === 2, "Adding product2 failed");

const total = calculateTotal(myCart);
console.assert(total === 1250, "Total calculation incorrect");

myCart = removeItemFromCart(myCart, "1");
console.assert(myCart["1"] === undefined, "Removing product1 failed");

const newTotal = calculateTotal(myCart);
console.assert(newTotal === 50, "New total calculation incorrect");

This complete example demonstrates how to define interfaces for data structures, create functions to manage data, and use assert statements to verify the correctness of the implementation. By leveraging TypeScript’s type system, we can build a more robust and maintainable shopping cart system. This also showcases a real-world example of using type aliases to improve code readability.

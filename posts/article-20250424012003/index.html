Building a Simple Grade Calculator in Python

Feb 29, 2024· Tutorial Island

Introduction

Today, we’re going to build a simple grade calculator using Python. This calculator will take a student’s scores on different assignments as input, calculate the average score, and then assign a letter grade based on the average. This exercise will give you practical experience in handling user input, processing data, and making decisions using conditional statements.

Explanation

Building a grade calculator presents several challenges:

  • Input Handling: We need to accept input from the user, which will likely be in the form of a string. We need to convert this string into numerical data that we can work with.
  • Data Validation: We should validate the input to ensure that the scores are within a valid range (e.g., 0-100).
  • Calculation: We need to calculate the average score from the list of individual scores.
  • Grading: We need to assign a letter grade based on the calculated average, using a predefined grading scale.

To address these challenges, we will leverage the following Python functionalities:

  • input() function: This function allows us to receive input from the user via the console.
  • str.split() method: This method allows us to split a string into a list of substrings based on a delimiter (e.g., a space or a comma).
  • int() and float() functions: These functions allow us to convert strings into integers and floating-point numbers, respectively.
  • try-except blocks: These blocks allow us to handle potential errors, such as invalid input (e.g., non-numeric values).
  • for loops: These loops allow us to iterate over a list of scores and perform calculations on each score.
  • if-elif-else statements: These statements allow us to make decisions based on the calculated average and assign the appropriate letter grade.

Requirements

Functional Requirements:

  • The program should accept a comma separated string of numerical scores from the user.
  • The program should calculate the average of the scores.
  • The program should assign a letter grade based on the following scale:
    • 90-100: A
    • 80-89: B
    • 70-79: C
    • 60-69: D
    • 0-59: F
  • The program should handle invalid input gracefully (e.g., non-numeric values, scores outside the 0-100 range).

Non-Functional Requirements:

  • The code should be well-documented and easy to understand.
  • The code should be efficient and avoid unnecessary computations.

Solutions

def validate_score(score):
    """Validates if a score is within the valid range (0-100)."""
    try:
        score = float(score)
        if 0 <= score <= 100:
            return score
        else:
            return None #Or raise ValueError depending on error handling preference
    except ValueError:
        return None

assert validate_score("85") == 85.0
assert validate_score("110") == None
assert validate_score("invalid") == None

The validate_score function ensures that the entered score is a valid number between 0 and 100. It first attempts to convert the input to a float. If the conversion fails, or if the number is outside the acceptable range, the function returns None. Returning None allows the calling function to handle the invalid score appropriately. We use float to allow for decimal scores.

def calculate_average(scores):
    """Calculates the average of a list of scores."""
    valid_scores = [score for score in scores if validate_score(score) is not None] #Filter valid scores

    if not valid_scores:
        return 0 #Avoid division by zero if there are no valid scores
    
    #Convert to float after validating
    valid_scores = [float(score) for score in valid_scores]

    return sum(valid_scores) / len(valid_scores)


assert calculate_average(["80", "90", "100"]) == 90.0
assert calculate_average(["invalid", "90", "110"]) == 90.0  #Handles invalid input
assert calculate_average([]) == 0 # handles empty lists

The calculate_average function takes a list of strings as input, validates each entry using the previous function, filters out invalid scores, converts the valid strings to float, and then calculates the average. The explicit conversion to float after validation prevents errors during the summation. It also includes a check to prevent division by zero if there are no valid scores. This function uses a list comprehension for a concise way to filter the score and create a new validated score list.

def assign_grade(average):
    """Assigns a letter grade based on the average score."""
    if 90 <= average <= 100:
        return "A"
    elif 80 <= average < 90:
        return "B"
    elif 70 <= average < 80:
        return "C"
    elif 60 <= average < 70:
        return "D"
    else:
        return "F"

assert assign_grade(95) == "A"
assert assign_grade(82) == "B"
assert assign_grade(55) == "F"

The assign_grade function takes the calculated average as input and assigns a letter grade based on the provided grading scale. The use of chained if-elif-else statements makes the logic clear and easy to follow.

Thoughts

We broke down the problem into smaller, manageable functions: validating input, calculating the average, and assigning the grade. This modular approach makes the code easier to understand, test, and maintain. By incorporating error handling and input validation, we ensure that the program is robust and can handle unexpected input gracefully.

Here’s the complete solution:

def validate_score(score):
    """Validates if a score is within the valid range (0-100)."""
    try:
        score = float(score)
        if 0 <= score <= 100:
            return score
        else:
            return None
    except ValueError:
        return None

def calculate_average(scores):
    """Calculates the average of a list of scores."""
    valid_scores = [score for score in scores if validate_score(score) is not None]

    if not valid_scores:
        return 0
    
    #Convert to float after validating
    valid_scores = [float(score) for score in valid_scores]

    return sum(valid_scores) / len(valid_scores)

def assign_grade(average):
    """Assigns a letter grade based on the average score."""
    if 90 <= average <= 100:
        return "A"
    elif 80 <= average < 90:
        return "B"
    elif 70 <= average < 80:
        return "C"
    elif 60 <= average < 70:
        return "D"
    else:
        return "F"

def main():
    """Main function to drive the grade calculator."""
    scores_input = input("Enter the scores separated by commas: ")
    scores = scores_input.split(",")

    average = calculate_average(scores)
    grade = assign_grade(average)

    print(f"The average score is: {average:.2f}")
    print(f"The letter grade is: {grade}")

if __name__ == "__main__":
    main()

assert validate_score("85") == 85.0
assert validate_score("110") == None
assert validate_score("invalid") == None

assert calculate_average(["80", "90", "100"]) == 90.0
assert calculate_average(["invalid", "90", "110"]) == 90.0  #Handles invalid input
assert calculate_average([]) == 0 # handles empty lists

assert assign_grade(95) == "A"
assert assign_grade(82) == "B"
assert assign_grade(55) == "F"

This complete solution combines all the individual functions into a working grade calculator. The main function handles the user input, calls the other functions, and prints the results. This approach allows for a clear and organized workflow, making the code easier to understand and maintain.
